Projects

For CoE Honor Code purposes most code is not public, can be made avaliable upon request.

B.O.T (Bringin' Outstanding Tunes)


Uses SpotiPy to check Spotify for new songs from a specified list of artist URIs. If the most recent song matches the current date, users will be sent album art, the track list, and a link to the song on spotify. Notifies through either GroupMe using groupyAPI or via text message using Twilio. Python script running as cronjob on an AWS EC2 instance.

Doppler Frequencies for GPS Satellites


Calculates the Doppler frequency from Gold codes given Gold code generator taps for each satellite out of 32 satellites. Simulation uses a Direct Sequence Binary Phase Shift Keying modulator at baseband. EECS 455: Wireless Communication Systems assignment using Matlab.

Log Manager


Utilizes unordered maps, stacks, and queues, in order to append and delete entries in excerpt list from master file of log entries. Searches based on category, keyword, timestamp, timestamp range. Emphasis on runtime and storage tradeoffs. A project in C++ for EECS 281: Data Structures and Algorithms.

Linker for LC2K


A Linker for the LC2K instructional set architecture. Takes assembled LC2K based object files and outputs machine code. Part of a series of LC2K projects for EECS 370: Computer Organization written in C; assembler, simulator, pipeline register, cache simulator.


LC2K Multiplier


A multiplier written in LC2K. Uses basic load, save, add, nor, and branch commands with a constraint on the number of operations. LC2K instructional set architecture used at University of Michigan, UC Santa Cruz, and University of Colorado.


Adaptive Volume


Uses a .wav input and microphone for ambient noise, adjusts the volume of the .wav file based on the volume of the outside noise. Written in Matlab as the final project for EECS 351: Digital Signal Processing.

NES Controller Interface


A sequential digital circuit on an Altera FPGA that interfaces with an NES game controller. Three signals, latch, pulse, and data, sampled at 60Hz with outputs written to LEDs and color sequences mapped to specific button press combinations. EECS 270: Logic Design written in Verilog.

Interactive Troubleshooting and Event Tracking


Interface using Java Swing framework deployed at GE Aviation Muskegon to 23 production Electrode Discharge Machines. Takes an input of machine problems and their respective solutions and generates step-by-step troubleshooting interface with instructions and pictures for maintenance workers. Also records solutions selected by user for later use in preventative maintenance.

Maze Navigation Robot


Two wheeled robot with multiple systems built including, computer vision webcam, heat detection thermopile with op-amp and ADC, sonar arduino sensor, wheel encoders. Built to locate object of specific temperature and color in a maze. Robot software written in C and Python, built for EECS 298: Electrical Engineering System Design final project. Our team won the award for the fastest object location time.

AM Radio Reciever


Composed of an amplifier, envelope detector and IF filter designed to receiver a song from a transmitter. A hardware assignment for EECS 216: Signals and Systems using a Keysight Oscilloscope, and DC power supply for the operational amplifier.

Breadboard Filters


Four basic RLC circuits designed in LTSpice and build on a breadboard in order to compare theoretical transfer functions to measured transfer functions. Used with multiple audio signals have noises at different frequencies. Assignment for EECS 215: Electronic Circuits.
